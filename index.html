A lightweight, easy to use, customizable Highscore Server.💾🎮

No login, no personal data required! Creating a leaderboard is as simple as just clicking a button. As it is made using sqlite which is local file based database, so it is somewhat faster than using an external database server. Some challanges that I faced while working with sqlite and how I tackled them - 1) Sqlite does not have support for auto time column update with new timestamps on an UPDATE query. So I have to write triggers to update time after each update. 2) Sqlite does not have OUTER JOIN or RIGHT JOIN. So, to perform an outer join I have to perform two Left Joins, once A with B and then B with A and then do a Union. But ultimately this was a fun learning experience. Read more. Try it now.
